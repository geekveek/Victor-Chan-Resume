User Instructions

Goal of the game

Get more points than the mouse before the time runs out to win. Earn points by killing the mouse.

Controls: Movement

WASD => Up, Left, Down, Right
Mouse => Look-At direction

Control: Weapon's Control

Left Mouse Button => Shoot
Spacebar/Right Mouse Button => Set trap

Artifacts

Crates => Collect resources for trap. Each crate contains enough materials for one trap.
Traps => Traps the mouse in the trap. The mouse becomes a vulnerable target.

Have fun playing the game!
